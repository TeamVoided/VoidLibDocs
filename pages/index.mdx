# Introduction
 
Welcome to VoidLib! VoidLib is a (hopefully) all-purpose FabricMC modding library

# What is VoidLib

As you've just read, VoidLib is an all-purpose FabricMC modding library.<br>
This means that if you want a feature that is not present you may request it over at [GitHub](https://github.com/TeamVoided/VoidLib)
